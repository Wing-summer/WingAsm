本软件仅提供基本的操作数高亮，如果需要寄存器高亮等请 PR 。

att 和 intel 这两个文件夹用于提供代码高亮的操作数，每行一个，空行忽略，如果行以 # 开头忽略。

masm 格式和 intel 应该大差不差，高亮这两者都使用 intel 的。

---------------------------------------------------------------------------------------

This software only provides basic operand highlighting. Please PR if register highlighting is needed.

The two folders att and intel are used to provide operands for code highlighting, one per line, blank lines are ignored, and lines starting with # are ignored.

The masm format should be similar to intel, and both of them use intel's highlighting.
